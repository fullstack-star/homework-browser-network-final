# Ajax失败后重试

```javascript
$.ajax({
    url : 'someurl',
    type : 'POST',
    contentType: "application/json",
    dataType: 'json',
    data :  {},   
    tryCount : 0,
    retryLimit : 3,
    timeout: 30000, // 超时时间 30 sec.
    success : function(json) {
        //do something
    },
    error : function(xhr, textStatus, errorThrown ) {
        if (textStatus === 'timeout') {
            this.tryCount++;
            if (this.tryCount <= this.retryLimit) {
                //try again
                $.ajax(this);
                return;
            }            
            return;
        }
        if (xhr.status === 500) {
            //handle error
        } else {
            //handle error
        }
    }
});
```
